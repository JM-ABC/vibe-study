# Vibe.Study — 바이브코딩 스터디

> Claude Code 기반 실전 바이브코딩 스터디 모집 페이지

## 소개

바이브코딩을 매일의 루틴으로 만드는 8주 스터디입니다.
매일 아침 30분 온라인(Google Meet) + 주 1회 잠실 오프라인으로 운영됩니다.

## 운영 방식

- **데일리 온라인**: 매일 30분 · Google Meet · 진행 공유 + AI 툴 팁
- **오프라인 모임**: 주 1회 · 잠실 카페 or 코워킹스페이스 · 데모 발표 + 실습
- **인원**: 6~10명 소수 정예
- **비용**: N빵 (장소·운영비 균등 분담)
